# React-Redux App Template
A boilerplate template for creating React/Redux applications.

Built with the following:
- [React v16](https://github.com/facebook/react)
- [Redux v4](https://github.com/reduxjs/redux)
- [Babel v7](https://github.com/babel/babel)
- [Webpack v4](https://github.com/webpack/webpack)
- [Jest v24](https://jestjs.io/docs/en/getting-started)

| File  | Purpose |
| ------------- | ------------- |
| [.eslintrc.js](.eslintrc.js)  | [ESlint](https://github.com/eslint/eslint) config settings  |
| [.gitignore](.gitignore) | Files and directories to ignore by Git  |
| [.npmrc]((.npmrc)) | Adds private NPM repository  |
| [.prettierrc](.prettierrc) | [Prettier](https://github.com/prettier/prettier) config settings  |
| [.stylelintrc.json](.stylelintrc.json) | [Stylelint](https://github.com/stylelint/stylelint) config settings and rules  |
| [babel.config.js](babel.config.js) | [Babel](https://babeljs.io/docs/en/configuration) presets and plugins  |
| [jest.config.js](jest.config.js) | [Jest](https://jestjs.io/docs/en/configuration) config settings and coverage rules  |
| [package-lock.json](package-lock.json) | Keeps track of NPM dependency tree  |
| [package.json](package.json) | Contains list of NPM dependencies  |
| [config/app-settings.js](config/app-settings.js) | Application constants and mock API server  |
| [config/webpack.config.babel.js](config/webpack.config.babel.js) | [Webpack](https://webpack.js.org/configuration/) config settings  |
| [config/webpack.optimization.js](config/webpack.optimization.js) | Minification and code splitting options  |
| [config/webpack.plugins.js](config/webpack.plugins.js) | Webpack plugins  |
| [config/webpack.rules.js](config/webpack.rules.js) | Webpack rules  |
